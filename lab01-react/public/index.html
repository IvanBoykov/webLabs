Ша́шки — логическая настольная игра для двух игроков, заключающаяся в передвижении определённым образом шашек по клеткам шашечной доски. Во время партии каждому игроку принадлежат шашки одного цвета. Цель игры — взять все шашки соперника или лишить их возможности хода (запереть).
